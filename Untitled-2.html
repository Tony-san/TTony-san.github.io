CSS Specificity

    CSS Specificity เป็นคอนเซปต์การเลือก CSS Selector ที่มีค่ามากที่สุดให้เหมาะสมเพื่อมาใช้กับ Element บนหน้า Browser

    ภาษา CSS เหมือนภาษาการเขียนโปรแกรมอื่น ๆ ที่อ่านโค้ดจากบนลงล่าง สรุปคือถ้าเขียน CSS Selector ซ้ำกัน 2 อัน อันล่างจะถูกนำไปใช้งานเพราะใหม่กว่า แต่ความพิเศษของ CSS คือ Selector ที่แตกต่างกันแต่ละ Selector จะมีค่าไม่เท่ากัน และเมื่อไหร่ที่ค่ามันต่างกัน ลำดับการเขียนโค้ดจะไม่มีผลทันที
